# 🤖 Бот-календарь в Telegram

Личный Telegram-бот, который следит за Google Календарём и Google Tasks:
рассказывает, что у тебя по делам, и заносит встречи/задачи прямо из переписки —
обычными человеческими фразами.

## План по шагам

- [x] **Шаг 1. Бот оживает** — отвечает тебе в Telegram. ← *мы здесь*
- [ ] **Шаг 2. Чтение календаря** — «что у меня сегодня / завтра / на неделе?»
- [ ] **Шаг 3. Запись в календарь** — добавление событий из Telegram.
- [ ] **Шаг 4. Умная понималка** — свободные фразы через Claude API.
- [ ] **Шаг 5. Задачи** — Google Tasks: галочки, срочность, бэклог без дедлайнов, автоперенос несделанного.
- [ ] **Шаг 6. Деплой на бесплатный PaaS** — бот онлайн 24/7, даже когда ноутбук выключен.

## Что нужно от тебя (по мере шагов)

1. **Токен бота** — создаётся за 2 минуты у [@BotFather](https://t.me/BotFather). *(Шаг 1)*
2. **Доступ к Google** — один раз разрешить через браузер. *(Шаг 2)*
3. **Ключ Claude API** — для умных фраз. *(Шаг 4)*
4. **Аккаунт на бесплатном PaaS** — чтобы выложить онлайн. *(Шаг 6)*

## Как создать токен бота (Шаг 1)

1. Открой в Telegram [@BotFather](https://t.me/BotFather).
2. Отправь `/newbot`.
3. Придумай **имя** бота (любое, например «Мой календарь»).
4. Придумай **username** — должен заканчиваться на `bot` (например `nastya_calendar_bot`).
5. BotFather пришлёт строку вида `123456789:AAH...` — это и есть токен.
6. Пришли этот токен мне (или впиши сам в файл `.env`, строка `TELEGRAM_TOKEN=`).

## Запуск локально (для разработки)

```bash
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
cp .env.example .env   # и заполнить TELEGRAM_TOKEN
python bot.py
```
